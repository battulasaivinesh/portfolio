OpenModelica GUI

A chemical modeling/simulation software used to simulate real-life chemical models. A graphical user interface was built on top of OpenModelica engine.

About The Project

A project developed as an intern under Deepak B Phatak, IIT Bombay. The main aim of this project was to build a Graphical User Interface for OpenModelica engine. At this project, I acted as the lead UI developer being the bridge between the users, chemical students, and the UI Development Team. The main challenge was designing the main canvas, as there were many components possible and we also had to manage the input and output connections of each model. After creating a complex model on canvas, we had to automatically generate code in the background using this info.
