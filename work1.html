CCA Application

CCA is the technical gymkhana of NIT Durgapur. This app was about all the current happenings in the club.


About The Project

A project developed as a Head, Web Design and Creative Team under Centre for Cognitive Activities, NIT Durgapur. The project was to develop an application where a user can get all the information about the club, and also know all the current happenings in the club. Time was the main challenge for this project, We completed this App in less than 2 weeks.
