Hell Ride

Hell Ride is a game built out of passion and love. It is a post-apocalyptic zombie game, where the hero has to survive and drive to the airport.


About The Project

A project developed as an Independent Game Developer under my own branding PRV Games. This was my first game, built with my passion for game development. This game was built using Unity and was only released for Android.
