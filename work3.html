Phaser Games

40+ games developed using PhaserJS. These games are used for educational purposes by Planet Spark.


About The Project

A project developed as an intern under Planet Spark. The project was to develop games on different concepts of Maths and Science. At this project, I acted as a Javascript Developer and built all the games using PhaserJS. The development time initially was around 10-15 days for one game. I developed a library on top of PhaserJS which helped to decrease the development time to 4 days. The main challenge was to optimize the performance of games, as the games were targetted for very low-end devices.
